### 1.7.4 break 和 continue
```py
In [33]: while True:
    ...:     name = input("Who are you?\n")
    ...:     if name != "Bob":
    ...:         continue
    ...:     print("Hello, Bob. What is your password?")
    ...:     password=input()
    ...:     if password == "fish":
    ...:         break
    ...: 
    ...: print("Access graned!")

Bob

fish

```
