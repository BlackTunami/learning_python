## 1.10 局部和全局作用域
全局作用：函数之外  
局部作用：函数之内  
局部可以访问全局变量，但是全局不能使用局部变量

局部可以访问全局变量，而全局不能使用局部变量

## 1.11 声明为全局变量global
```py
def spam():
    global eggs 
    eggs = 5


eggs=7
spam() # 调用spam函数
print(eggs) 
5
```


```py
def spam():
    global eggs #全局变量
    eggs="spam"


def bacon():
    eggs="bacon" #局域变量


def ham():
    print(eggs) #全局变量


eggs=42
spam()
print(eggs)
spam
```
